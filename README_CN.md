# Neovim 配置 中文版

## 依赖
### [coc.nvim](https://github.com/neoclide/coc.nvim)
`coc.nvim` 是一个轻量化的**代码补全框架**，它不仅提供很多方便使用的函数，还支持多种语言服务，可以满足各种语言的补全功能。由于它的后端使用`nodejs`实现，所有你需要安装`nodejs`，另外还需要`yarn`来下载`其他的coc插件`。
#### ArchLinux 安装
```sh
pacman -S nodejs yarn
```
